Select and merge [github/gitignore](https://github.com/github/gitignore).

To use, you have to clone [github/gitignore](https://github.com/github/gitignore) repository.

## Usage

`java -jar gitignore_merger.jar /PATH/TO/gitignore LANGUAGE1 LANGUAGE2...`
